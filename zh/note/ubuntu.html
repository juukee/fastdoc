  • 更新系统

    • xsession配置
      • 这一点至关重要,重启xrdp
        • 可能出现的问题（默认浏览器无法使用），安装firefox浏览器或者Chrome浏览器

    # 更新系统

    sudo apt update -y && sudo apt upgrade -y
    1
    2

    # 安装xfce4界面

    sudo apt-get install xfce4 -y
    1

    # 安装window xrdp模块

    sudo apt-get install xrdp -y
    1

    # xsession配置

    对于非root用户登录服务器的情况，应该首先将xfce4-session写入该用户文件夹的xsession文件：/home/username/.xsession echo "xfce4-session" >~/.xsession

    # 这一点至关重要,重启xrdp

    sudo service xrdp restart
    1

    # 可能出现的问题（默认浏览器无法使用），安装firefox浏览器或者Chrome浏览器

    sudo apt install firefox
    1
